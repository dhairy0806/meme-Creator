# 😂 Meme Generator

AI-powered meme generator website!

## 🚀 Quick Start

1. Download all files
2. Open `index.html` in browser
3. Enter topic → Generate → Download!

## ✨ Features
- ✅ 3+ meme templates
- ✅ Auto funny captions
- ✅ One-click download
- ✅ Mobile responsive
- ✅ No server needed
